Задачей агента поиска всех нерассмотренных предложений является поиск всех нерассмотренных предложений. У данного агента нет входных параметров. Агент инициируется при условии появления в памяти вопросной конструкции, соответствующей действию. найти все нерассмотренные предложения. Возможные результаты работы агента:

  • Если в результате поиска нерассмотренные предложения не были найдены, то генерируется сообщение "The search hasn't given any results".
  • Если в результате поиска нерассмотренные предложения были найдены, то выводятся все найденные предложения. Нерассмотренное предложение - предложение, которое не утверждено и не отклонено. Также действие, к которому относится предложение, должно быть инициировано. Класс действия должен являться ключевым sc-элементом Раздел. Предметная область действий разработчиков sc-моделей баз знаний, соответствующих методике, предполагающей взаимодействие всех разработчиков с семантической памятью и наличием иерархии администраторов. Если класс действия не является ключевым sc-элементом данного раздела, то данный класс действия не будет добавлен в результат работы агента.
