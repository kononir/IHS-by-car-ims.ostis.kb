Задачей агента вывода изменений заданной sc-структуры является вывод изменений заданной структуры с позиции выполненных действий. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу вывода изменений заданной структуры. Единственным аргументом запроса является раздел базы знаний, содержащий спецификации выполненных пользователем действий.
