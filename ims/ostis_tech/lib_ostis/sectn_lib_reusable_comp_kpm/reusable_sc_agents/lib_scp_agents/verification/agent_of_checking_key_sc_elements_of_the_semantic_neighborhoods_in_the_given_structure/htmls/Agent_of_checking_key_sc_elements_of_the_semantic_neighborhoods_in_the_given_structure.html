Задачей агента проверки указания ключевых sc-элементов семантических окрестностей в рамках заданной структуры является нахождение семантических окрестностей, у эелементов заданной структуры, у которых не указан ключевой sc-элемент. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки указания ключевых sc-элементов семантических окрестностей в рамках заданной структуры. Единственным аргументом запроса является структура, элементы которой будут проверены на наличие семантической окрестности. Возможные результаты работы агента:

  • Если на вход агенту в качестве первого параметра пришла не структура, то генерируется сообщение об ошибке (Wrong 1st parameter! It must be a structure.).
  • Если в рамках заданной структуры были найдены узлы, у которых не указан ключевой sc-элемент семантической окрестности, то они заносятся в структуру. не указан ключевой sc-элемент семантической окрестности .
  • Если в результате работы агента такие узлы не найдены, то генерируется сообщение об отсутствии данного рода ошибок в рамках заданной структуры(Problematic knowledge base fragments are not found.).
