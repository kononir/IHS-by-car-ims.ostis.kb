Задачей агента наличия максимального класса объектов исследования у предметных областей является поиск предметных областей из предложенного контура, для которых не указан максимальный класс исследования. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу проверки наличия максимального класса объектов исследования у предметных областей. Единственным аргументом запроса является знак структуры, элементы которой проверяются. При необходимости ограничения области видимости агента в качестве второго аргумента запроса подается контекст действия. Возможные результаты работы агента:

  • Если на вход агенту в качестве первого параметра пришла не структура, то генерируется сообщение об ошибке (Wrong 1st parameter! It must be a structure.).
  • Если контекст действия, поданный на вход агенту в качестве второго аргумента, не является структурой, то генерируется сообщение об ошибке (Wrong 2nd parameter! It must be a structure.).
  • Если в результате работы агента были найдены классы, для которых не указан максимальный класс исследования, то они заносятся в структуру. не указан максимальный класс объектов исследования предметной области .
  • Если в результате работы агента такие классы не найдены, то генерируется сообщение об отсутствии такого рода ошибок в рамках заданной структуры (Problematic knowledge base fragments are not found.).
