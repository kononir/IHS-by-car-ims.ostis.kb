Задачей scp-программа посика семантической окрестности действияявляется добавление во множество конструкций, являющихся спецификациями этих инициированных действий. Первым входным параметром является множество, элементами которого являются инициированные действия. Вторым входным параметром является множество, которое будет содержать спецификации действий, содержащихся во множестве, поданном в качестве первого аргумента. В ходе выполнения программы явно указывается связь со следующими ключевыми узлами: абсолютными понятием инициированное действие, неролевыми отношениями исполнитель*, предложение* и автор*.
